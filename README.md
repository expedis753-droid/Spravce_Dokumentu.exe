# Spravce Dokumentu

Aplikace pro spravu dokumentu s online databazi.

## Stazeni

Stahni soubor `Spravce_Dokumentu.exe` z [Releases](https://github.com/expedis753-droid/Spravce_Dokumentu.exe/releases).

## Funkce

- Prihlaseni/registrace uzivatelu
- Kategorie dokumentu
- Nahravani souboru (TXT, PDF, MD, kod)
- Vyhledavani v dokumentech
- Statistiky (radky, slova, znaky)
- Tagy, poznamky, propojeni dokumentu
- Online databaze - sdileni mezi uzivateli
